ANLEITUNGEN & TOOLTIPS (SETUP)
--------------------------------------------------------------------------------

A: URL der Online-Anmeldung (*)
Bitte gib hier den Link zur verknüpften Google-Tabelle an, in der die Antworten deiner Freizeitanmeldung gesammelt werden.
Wichtig: Kopiere nicht den Link zum öffentlichen Anmeldeformular, sondern die URL der dazugehörigen Antwort-Tabelle (Sheets), auf die du als Administrator Zugriff hast.

B: Art der Datenerfassung - Prüfkriterium (**)
Lege hier fest, wie das System mit den Daten arbeiten soll:
„Ja“: Die Daten werden automatisch aus der verknüpften Tabelle bezogen.
„Nein“: Die Daten werden von dir manuell eingetragen.

C: Erstmalige Zugriffs-Bestätigung (***)
Damit Google Sheets Informationen aus der externen Datenbank (z. B. für Landkreise und Orte) laden kann, musst du die Verbindung einmalig autorisieren:
Falls eine Meldung wie #REF! oder „Berechtigung erforderlich“ erscheint, klicke auf die entsprechende Zelle. Klicke im erscheinenden Fenster auf den blauen Button „Zugriff zulassen“.
Hinweis: Dies ist ein Sicherheitsmechanismus von Google, der nur einmal pro Datei aktiviert werden muss.

D: Anzeige-Modus - Debug & Drucken (****)
Wähle aus, wie die Liste und eventuelle Fehler dargestellt werden sollen:
- Normal: Die Liste wird berechnet. Kritische Fehler (z. B. „Zu wenige Teilnehmende“) erscheinen als Texthinweis.
- Debug: Zeigt zusätzlich einen Diagnose-Block über der Liste an (z. B. Input-Zeilen, Filter-Statistiken, Berechnungsdauer). Ideal zur Fehlersuche.
- Drucken (Clean Mode): Unterdrückt alle Fehlermeldungen. Ist eine Liste ungültig, bleibt sie komplett leer. So wird verhindert, dass versehentlich Fehlertexte mit ausgedruckt werden.

E: Audit-Modus (*****)
Der Audit-Modus deaktiviert die automatische Filterung. Er zeigt stattdessen alle angemeldeten Personen an – inklusive einer Begründung, warum jemand ausgeschlossen wurde (z. B. „Alter“ oder „Wohnort“).
Nutzen: Verwende diesen Modus zur Fehlerkontrolle, um transparent nachzuvollziehen, warum bestimmte Personen nicht auf der offiziellen Zuschussliste erscheinen.
